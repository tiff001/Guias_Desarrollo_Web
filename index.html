Información de la materia

Materia: Desarrollo Web

Docente: Ing. Guillermo Calderón

Descripción: El curso de Desarrollo Web nos introduce al mundo de la creación y diseño de sitios y aplicaciones web, combinando lo técnico con lo creativo. Su objetivo es que aprendamos a construir páginas funcionales, visualmente atractivas y adaptables a distintos dispositivos, utilizando HTML, CSS y JavaScript.

Durante el ciclo, aprendimos a estructurar el contenido con HTML, dar estilo y diseño con CSS, usando herramientas como el modelo de caja, Flexbox y CSS Grid, y aplicar Bootstrap para crear interfaces modernas y responsivas. Finalmente, exploramos JavaScript, comprendiendo cómo añadir interactividad y dinamismo a las páginas web.

Contenidos de la materia

  • HTML: estructura y semántica de páginas web.
  • CSS: estilos, colores, tipografías, modelo de caja, Flexbox y Grid.
  • Maquetación: diseño de layouts y sitios responsivos.
  • Bootstrap: uso de componentes y utilidades para diseño moderno.
  • JavaScript: sintaxis básica, funciones, eventos y objetos.

Guías

Guía 1

  • Ejemplo 1 - Estilos embebidos
  • Ejemplo 2 - Poema
  • Ejemplo 3 - Países
  • Ejemplo 4 - Serie favorita

Descripción de la guía:

La guía se centró en los conceptos básicos de HTML y CSS, con el propósito de aprender a estructurar una página web y aplicar estilos visuales. A través de tres ejercicios, se exploraron las distintas formas de incluir CSS en un documento HTML y cómo usar selectores, propiedades y reglas para dar formato al contenido.

Guía 2

  • Ejemplo 1 - Película
  • Ejemplo 2 - Tablero
  • Ejemplo 3 - Biografía

Descripción de la guía:

La guía se enfocó en el uso de selectores CSS avanzados, pseudo-clases, pseudo-elementos y el modelo de caja, con el objetivo de aplicar estilos más precisos y dinámicos en las páginas web. Se aprendió a seleccionar elementos específicos, controlar su apariencia y diseñar estructuras visuales con mayor control y creatividad.

Guía 3

  • Ejemplo 1 - Premios Oscars
  • Ejemplo 2 - Menú horizontal
  • Ejemplo 3 - Viajes
  • Ejemplo 4 - Posicionamiento

Descripción de la guía:

La guía se centró en el posicionamiento, visualización y maquetación de sitios web, aplicando las propiedades de CSS para organizar correctamente los elementos en pantalla y crear estructuras de diseño tanto fijas como fluidas. Se aprendió a manejar distintos tipos de posicionamiento, controlar la visualización de los elementos y maquetar páginas completas con diferentes técnicas.

Guía 4

  • Ejemplo 1 - Uso de Media Queries
  • Ejemplo 2 - Layout responsivo

Descripción de la guía:

La guía se enfocó en la introducción al diseño web responsivo, aplicando los conocimientos de HTML5 y CSS3 para crear páginas que se adaptan correctamente a distintos tamaños de pantalla, desde computadoras hasta tablets y teléfonos móviles. Se aprendió a usar la meta-etiqueta viewport, las media queries y las unidades relativas para lograr estructuras fluidas y flexibles.

Guía 5

  • Ejemplo 1 - Introducción a JavaScript

Descripción de la guía:

La guía introdujo los conceptos fundamentales de JavaScript, su sintaxis y formas de integración en HTML. Se aprendió a generar contenido dinámico, manipular el DOM y crear interacciones mediante funciones como alert(), prompt() y confirm(). También se exploró el uso de atributos como defer y async para optimizar la carga de scripts, así como la depuración de errores desde la consola del navegador.

Guía 6

  • Ejemplo 1 - Index
  • Ejemplo 2 - Index con funciones

Descripción de la guía:

La guía presenta los conceptos esenciales de las funciones en JavaScript, incluyendo su declaración, uso de parámetros, funciones anónimas y funciones flecha. También explica el manejo de eventos para interactuar con elementos HTML. En la parte práctica, se crean páginas que usan funciones para procesar datos, generar tablas, aplicar recursividad y manejar formularios con validación básica.

Guía 7

  • Ejemplo 1 - Index

Descripción de la guía:

La guía introduce el uso del DOM para acceder, modificar, crear y eliminar elementos dentro de una página web. Explica la estructura del documento como un árbol de nodos, los tipos de nodos, y los métodos modernos y tradicionales para seleccionar elementos. En la parte práctica, se crean formularios dinámicos, se recorren estructuras del DOM, se manipulan estilos y se generan páginas completas mediante JavaScript.
